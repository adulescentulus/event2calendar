    Beschreibung

    DevOps is not only about automation, continuous integration and delivery processes, but also about people. People play a crucial role in DevOps success, and they are even more relevant when we introduce security in DevOps routines. Implementing security with the help of tools only into DevOps not always works. After applying the coolest and highest security solutions in the pipeline, work only starts. Developers play with them, but for different factors like prioritization, knowledge gap, lack of support, and so on, we stop using them. And fall back into old routines. So how to achieve DevSecOps culture? To create a security culture, we need to start with behavioural changes. And here comes the trick.The talk is all about trials, failures and success of achieving security culture in DevOps oriented organization, making security part of DevOps by shaping people behaviour. Come to the session to learn from others experience, failures and stories what tricks and activities can be used to get security successfully in a place.
